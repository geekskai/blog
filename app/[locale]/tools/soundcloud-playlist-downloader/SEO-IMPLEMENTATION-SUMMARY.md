## üîß Schema.org Validation Checklist

### ‚ö†Ô∏è Common Schema Validation Errors & Solutions

#### 1. Missing `aggregateRating` Field (Fixed: 2026-01-02)

**Problem**:

- Google Rich Results Test may show warnings about missing `aggregateRating` field for `WebApplication` or `SoftwareApplication` types.

**Solution**:
Always include `aggregateRating` in your WebApplication/SoftwareApplication schemas:

```typescript
aggregateRating: {
  "@type": "AggregateRating",
  ratingValue: "4.8",        // String format required
  ratingCount: "1250",        // String format required
  bestRating: "5",            // Usually "5"
  worstRating: "1",           // Usually "1"
}
```

**Files to Check**:

- ‚úÖ `app/[locale]/tools/[tool-name]/layout.tsx` - WebApplication schema
- ‚úÖ `app/[locale]/tools/layout.tsx` - ItemListElement SoftwareApplication schemas
- ‚úÖ Any other layout files with WebApplication/SoftwareApplication types

**Validation Tools**:

- [Google Rich Results Test](https://search.google.com/test/rich-results)
- [Schema.org Validator](https://validator.schema.org/)
