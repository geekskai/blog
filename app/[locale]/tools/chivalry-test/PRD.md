# Chivalry Test Tool – Product Requirements Document

## 1. Overview

The **Chivalry Test** is an interactive online quiz designed to help users evaluate how chivalrous they are in modern society.  
It combines psychology-style self-assessment with contemporary social scenarios, offering a fun, shareable, and insightful experience.

This tool is primarily built for:

- Organic SEO traffic acquisition
- Social sharing and viral growth
- User engagement and brand awareness

---

## 2. Goals & Objectives

### Primary Goals

- Capture organic traffic from the keyword **“chivalry test”** and related long-tail queries
- Increase average session duration on the website
- Encourage social sharing of test results

### Secondary Goals

- Build topical authority around personality tests and lifestyle quizzes
- Drive internal traffic to other tools on the website
- Collect anonymous usage insights (non-PII)

---

## 3. Target Users

- Men aged 18–45
- English-speaking users
- Interested in self-improvement, dating, personality tests, or fun quizzes
- Social media users who enjoy sharing test results

---

## 4. Core Features

### 4.1 Quiz System

- Total Questions: 10–15
- Question Types:
  - Multiple choice (single select)
- Question Topics:
  - Manners and politeness
  - Respect toward others
  - Responsibility and integrity
  - Modern dating behavior
  - Social awareness

Example Question:

> If you're on a date and your phone rings, what do you do?

---

### 4.2 Scoring Logic

- Each answer has a weighted score
- Final score range: **0–100**
- Score calculated instantly upon completion

---

### 4.3 Result Levels

| Score Range | Title                | Description                       |
| ----------- | -------------------- | --------------------------------- |
| 0–30        | The Anti-Chivalrist  | Chivalry is not your thing        |
| 31–60       | The Casual Gentleman | You have basic manners            |
| 61–85       | The Modern Knight    | You value respect and integrity   |
| 86–100      | The True Gentleman   | Chivalry is part of your identity |

Each result includes:

- Title
- Short personality description
- Strengths
- One improvement suggestion

---

### 4.4 Share & Engagement

- Share buttons:
  - Twitter (X)
  - Facebook
  - WhatsApp
  - Copy link
- Auto-generated share text:
  > "I scored 78 on the Chivalry Test – I'm a Modern Knight. How chivalrous are you?"

## 6. UX & UI Requirements

- Mobile-first design
- One question per screen
- Progress indicator (e.g. 3 / 12)
- Instant result display (no reload)
- Clean, neutral, modern visual style

---

## 7. Technical Requirements

- Frontend: React / Vanilla JS
- No login required
- No personal data collection
- Fast load time (<1s on mobile)
- Fully client-side scoring
