## 💻 如何使用

### 1. 本地开发
```bash
# 启动开发服务器
npm run dev

# 访问页面
http://localhost:3000/love
```

### 2. 自定义内容
详见 `USAGE.md` 文件，包含：
- 修改名字教程
- 修改表白内容

npm run build
