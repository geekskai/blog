## 🎓 使用指南

### 基础使用
1. 访问页面，享受自动烟花秀
2. 点击"点击放烟花 ✨"按钮，触发8枚烟花
3. 点击页面任意位置，爆发爱心特效
4. 观看浪漫语录轮播
5. 长按页面5次，解锁隐藏彩蛋

### 自定义内容
编辑 `page.tsx`:

```typescript
// 修改名字
<h1>你的名字</h1>

// 修改表白内容
<div>
  你想说的话...
</div>

// 添加音乐
<MusicPlayer src="/your-music.mp3" />
```
