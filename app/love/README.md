## ✨ 特色功能

### 🎆 视觉效果
- **烟花动画**: Canvas 绘制的华丽烟花效果，支持手动触发
- **粒子背景**: 流动的粒子网络，营造梦幻氛围
- **星空背景**: 120+ 闪烁的星星
- **浮动花瓣**: 随机飘落的表情符号（花朵、爱心等）
- **3D卡片**: 鼠标移动时的 3D 倾斜效果
- **自定义光标**: 粉红色渐变光标，增强交互体验

### 🎨 设计亮点
- **玻璃态设计**: 卡片使用毛玻璃效果（backdrop-blur）
- **渐变色彩**: 粉红、紫色渐变主题
- **发光效果**: 文字和元素的多层阴影发光
- **动画细节**: 心跳、浮动、脉冲等精心设计的动画

### 💻 技术实现

#### Canvas 动画
1. **烟花系统**
   - 火箭发射与爆炸
   - 粒子轨迹系统
   - 重力和衰减效果
   - 多彩渐变色彩

2. **背景粒子**
   - 80个浮动粒子
   - 粒子间连线
   - 边界循环处理

#### React Hooks
- `useRef`: Canvas 引用和数据持久化
- `useCallback`: 性能优化，避免重复创建函数
- `useEffect`: 动画循环和事件监听管理
- `useState`: 卡片显示状态控制

#### 性能优化
- RequestAnimationFrame 动画循环
- Canvas 双层渲染（背景 + 烟花分离）
- 及时清理事件监听器和动画帧
- 粒子数量和更新频率优化

## 🎯 使用方式

### 访问页面
```
/love
```

### 自定义内容
修改 `page.tsx` 中的以下内容：
- 名字: 搜索 "陈成连" 并替换
- 表白文字: 修改卡片中的文本内容
- 颜色主题: 修改 `COLORS` 数组

### 自定义动画
```typescript
// 修改粒子数量
for (let i = 0; i < 80; i++) // 背景粒子数量

// 修改烟花频率
if (frameCountRef.current % 80 === 0) // 每80帧发射一次

// 修改花瓣生成间隔
const interval = setInterval(createPetal, 600) // 600ms一次
```
